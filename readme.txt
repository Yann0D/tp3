# Mon Projet

Ceci est un projet démo pour présenter l'utilisation de Git et GitHub.

## Description

Ce projet contient un programme "Hello World" en Java.

## Fonctionnalités

- Affiche "Hello, World!" à l'écran.

## Installation

1. Clonez ce dépôt sur votre machine locale.
2. Assurez-vous d'avoir Java installé sur votre système.
3. Compilez le fichier HelloWorld.java en utilisant la commande `javac HelloWorld.java`.
4. Exécutez le fichier de classe généré en utilisant la commande `java HelloWorld`.
